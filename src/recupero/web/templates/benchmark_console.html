🎯 Trace Benchmark

Trace-accuracy score for a single case. Paste your operator key and a case id to load; data is fetched directly from the admin-gated API (the key stays in a request header, never the URL).
Scores the case's already-produced trace against a STORED ground_truth.json — recall (did we reach the known endpoints?), endpoint precision (were our flagged endpoints real?), and F1. This console never recomputes the trace and never fabricates numbers — it displays only the scored result.
⚠ GROUND TRUTH MUST BE INDEPENDENTLY VERIFIED The ground truth is operator-supplied input from an independent, verified source (a published incident report, an indictment, a recovered-funds record). It is NEVER derived from the trace itself — that would be circular.
