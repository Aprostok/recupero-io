📋 Case Overview

Per-case launcher. Paste your operator key and a case id to see which deliverables the case has produced, with a deep link into each one. Data is fetched directly from the admin-gated API (the key stays in a request header, never the URL).
Read-only, filesystem-only: deliverables are detected by file presence on disk — this console never parses case.json or any artifact, so it stays robust against a corrupted case.
