📑 Exhibit Pack

Court-admissible exhibit index for a single case. Paste your operator key and a case id to load; data is fetched directly from the admin-gated API (the key stays in a request header, never the URL).
Each case artifact is listed as Exhibit A, B, C… with its SHA-256 hash and byte size — so a court / opposing expert can confirm the file introduced into evidence is byte-identical to what was produced.
