⚖ Law Firm Dashboard

Per-law-firm portfolio rollups across every referred case: volume (cases referred, traces completed, cases in queue, cases with letters sent), money (total loss, total frozen, total returned to victims), throughput medians (intake → first letter, letter → first freeze), and the top issuers seen across the firm’s caseload. Deployment-wide, sourced from the DB. Paste your operator key to load; data is fetched directly from the admin-gated API (the key stays in a request header, never the URL).
